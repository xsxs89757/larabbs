## 1.安装laravel 应用

### 1)composer 安装 切换至packagist
#### composer config -g repo.packagist composer https://packagist.laravel-china.org
### 2)laravel 应用安装
#### composer create-project laravel/laravel larabbs --prefer-dist "5.5.*"
